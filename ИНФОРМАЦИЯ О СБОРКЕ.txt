в проекте может быть использована полная комплектация Bootstrap4, просто нужно будет подключить все компоненты B4 
через импорт (в данный момент к проекту подключена только сетка Bootstrap4)

работа над проектом:
(перед началом работы удалить папку node_moduls, будем устанавливать заново свежие компоненты в проект)
1. в корневой папке проекта вызвать PowerShell, ввести команду ncu (проверка последних версий компонентов) - покажет какие компоненты нужно обновить
2. обновляем компоненты командой ncu -u
3. установим все компоненты в проект командой npm i
4ю можно начинать работу с проектом

-------

файл "ht.access" при загрузке на хостинг позволит кэшировать данные проекта, что ускорит загрузку некоторых элементов на странице (картинок и т.п.)
